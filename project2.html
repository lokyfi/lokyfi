Project 2 Report: Interactive Portfolio with AI Chatbot

Project Overview

This project enhances a personal portfolio website by integrating an AI-powered chatbot built using Chatbase. The chatbot answers questions about work, projects, and skills, making the portfolio more interactive and accessible.

Project Requirements Document (PRD)

Problem Statement

Static portfolios often lack interactivity. Visitors may not engage deeply or might miss key parts of the content. A chatbot can guide users and offer tailored answers, improving engagement and accessibility.

Documentation

1. Introduction

This project enhances a portfolio with an AI chatbot to improve visitor engagement. Chatbase was chosen for its ease of use and ability to train the bot on specific content. The target audience includes potential employers, collaborators, and anyone interested in the work.

2. Implementation Process

Content Preparation: Carefully curated portfolio content was organized into logical sections before uploading to Chatbase.

Integration: The Chatbase widget was easily embedded into the existing HTML portfolio with just a few lines of JavaScript code.

Challenges: The free plan has limitations on the number of messages, and initial response accuracy required several iterations of training.

3. Extensions Implemented

Multi-language Support: Added Russian language responses to make the portfolio accessible to a wider audience.

Custom Welcome Message: Created a friendly, personalized greeting that matches the portfolio's tone.

4. User Testing

Tests were conducted with 5 users who asked a variety of questions. Based on their feedback, the bot's responses to common questions were improved and its personality was adjusted to be more professional yet approachable.

5. Conclusion

This project successfully added an interactive element to the portfolio. Key lessons included the importance of carefully structuring training data and the value of user testing. Future improvements could include adding more specialized knowledge and implementing an analytics dashboard.
